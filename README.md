# Multidimensional Data Processing - Exercise Set

## Description
A collection of exercises completed as part of laboratory classes.  
Each lab directory focuses on a specific programming concept and provides a simple, self‑contained example.

## Contents
- Individual exercises placed in separate directories (each directory contains 2–3 tasks)  
- Minimal implementations  
- Due to licensing constraints, only exercises without original exercies descriptions are included  

## Technologies Used
- Python  
- NumPy  
- SciKit‑Learn  
- SciPy  

## Purpose
This repository is intended for educational use and serves as a quick reference while learning and practicing programming concepts.
